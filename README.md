# Отпуска сотрудников

Готовая статическая страница для GitHub Pages.

Файлы:
- index.html

Особенности:
- работает без сборки;
- данные сохраняются в браузере через localStorage;
- есть импорт/экспорт JSON;
- пароль администратора: admin123.

Чтобы поменять пароль, откройте index.html и замените строку:

const ADMIN_PASSWORD = "admin123";
