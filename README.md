# Vacation Tracker — GitHub-only

Файлы:
- `index.html`
- `data.json`
- `README.md`

Сайт хранит общие данные в `data.json` и сохраняет изменения через GitHub API.

Важно: нужен GitHub fine-grained token с доступом к этому репозиторию:
- Contents: Read and write
- Metadata: Read-only
